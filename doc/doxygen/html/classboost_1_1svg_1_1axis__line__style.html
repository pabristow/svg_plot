Detailed Description

Style of the X or Y-axes lines.

(But NOT the ticks and value-labels because different styles for X and Y-axes are possible).

Parameters
dX or Y axis.
minMinimum of axis line.
maxMaximum of axis line.
colAxis line color.
widthAxis line width.
axis_positionHow the axes intersect with values as below:
enum x_axis_intersect {bottom = -1, x_intersects_y = 0, top = +1}; enum y_axis_intersect {left = -1, y_intersects_x = 0, right = +1}; If axes look like an L, then is bottom left. If a T then y intersects and X is at bottom.
label_onLabel axis with text - example: "length".
label_units_onLabel axis units, example: "cm".
axis_lines_onDraw an X horizontal or a Y vertical axis line.
axisDepending on value of dim, either X-axis (y = 0) transformed into SVG Y coordinates, or Y-axis (x = 0) transformed into SVG X coordinates (-1 if not calculated yet).
