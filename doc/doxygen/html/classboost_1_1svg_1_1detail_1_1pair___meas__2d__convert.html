Detailed Description

template<bool correlated>
class boost::svg::detail::pair_Meas_2d_convert< correlated >

This functor allows any 2D data convertible to type std::pair<Meas, unc> to be plotted.

Template Parameters
correlatedtrue if the uncertainties are correlated (for example, adding to a constant value).
