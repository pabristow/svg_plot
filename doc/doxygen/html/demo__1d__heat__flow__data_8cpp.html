Detailed Description

Demonstration of 1D plotting with some real-life data.

An example to show Heat Flow data from http://www.itl.nist.gov/div898/handbook/eda/section4/eda4281.htm This data set was collected by Bob Zarr of NIST in January, 1990 from a heat flow meter calibration and stability analysis.

This data was used in an example in the Boost.Math Library.
