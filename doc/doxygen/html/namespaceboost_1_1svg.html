Inspect values to find min and max.

Inspect all values between begin and (one before) end to work out and update min and max. Similar to boost::minmax_element, but ignoring at 'limit': non-finite, +-infinity, max & min, & NaN). If can't find a max and a min, then throw a runtime_error exception.

Template Parameters

Parameters
beginIterator to chosen first item in container.
endIterator to chosen last item in container.
minUpdated with Minimum value found (not 'at limit').
maxUpdated with Maximum value found (not 'at limit').
Returns
