◆ fp_type

typedef double boost::svg::fp_type

Allows a switch between using double or float to hold many floating-point plotable data values as either double or float. (32-bit has sufficient precision for data plots, so using float might be faster and/or take less space, but with a smaller range of (std::numeric_limits<>::max)() to (std::numeric_limits<>::min)(), about 10^38 compared to 64-bit 10^308).
