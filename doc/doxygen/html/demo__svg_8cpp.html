Detailed Description

Demonstration of a few facets of using the SVG interface directly.

This only demonstrates a very few of the possible features because most users will only need to use the plot interfaces. But this might provide a little guidance for producing other diagrams and drawings. This module is to demonstrate features of the svg class. It is entirely contrived and has no other conceivable use!
