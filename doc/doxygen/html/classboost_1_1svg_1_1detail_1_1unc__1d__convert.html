Detailed Description

template<bool correlated>
class boost::svg::detail::unc_1d_convert< correlated >

This functor allows any 1D data that can be converted to boost::quan::unc (uncertain double) to be plotted.

Defaults provided by the unc class constructor ensure that uncertainty, degrees of freedom information, and type are suitably set too.
