Detailed Description

Demonstration of showing the 1D values.

Showing the 1D values of items from the data set.

Some of the many possible formatting options are demonstrated, including controlling the precision and iosflags, and prefix and suffix also useful for giving units.

Quickbook markup to include in documentation.
