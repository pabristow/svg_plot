Detailed Description

Create box plots in Scalable Vector Graphic (SVG) format.

Provides svg_boxplot data and functions to create plots, and svg_boxplot_series to allow data values to be added to the boxplot.
Very many functions allow fine control of the appearance and layout of plots and data markers. (Items common to 1D, 2D and boxplot use axis_plot_frame).

A convenient way of graphically depicting groups of numerical data through their five-number summaries.
Show 1st quartile, median and 3rd quartile as a box. http://en.wikipedia.org/wiki/Boxplot
