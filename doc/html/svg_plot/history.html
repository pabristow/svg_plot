History and Status

History

This project started as a Google Summer of Code project supporting Jake Voytko in 2007.

It was used during development to produce plot of mathematical functions for the Boost.Math library, and it still in use now.

Over the years since it has been enhanced, in particular to handle measurements and observations of uncertain data.

In 2021, a major refactoring and reburbishment has been done, both to documentation and the code. Numerous bugs have been squashed, new features added (Unicode data-point markers) and more examples added.

The repository is now Github repository SVG_plot.

Status
Important

This is NOT an official Boost library : it was proposed as a library but was rejected as being out-of-scope for Boost. It was a Google Summer of Code project (2007) whose mentor organization was Boost. It remains a library under construction, the code is quite functional, but interfaces, library structure, and names may still be changed without notice.
