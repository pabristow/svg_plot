Function quantile

boost::svg::quantile

Synopsis

// In header: <boost/svg_plot/quantile.hpp>


double quantile(vector< double > & data, double p, int HF_definition = 8);

Description

Estimate quantile from values in vector data.

Note

Assumes values are pre-sorted from min to max.

Parameters:

HF_definition

Algorithm to use for the estimation.

data

Population for which to estimate quantile (percentile). Data must be ordered minimum to maximum.

p

Fraction of population, for example, p = 0.25 for 1st quartile. (Usually p = 0.25, or p = 0.75 for boxplots).

Returns:

Estimated quantile from the population.
