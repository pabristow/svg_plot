Function default_font

boost::svg::default_font

Synopsis

// In header: <boost/svg_plot/svg_style.hpp>


const char * default_font("Lucida Sans Unicode");

Description

Default font chosen is a Unicode font like ['Lucida Sans Unicode] that has the best chance of ['symbols] being rendered corrrectly. Used for title, legend, axes ... unless overridden by an explicit font specification.
