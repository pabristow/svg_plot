Function operator<<

boost::svg::operator<<

Synopsis

// In header: <boost/svg_plot/svg_style.hpp>


std::ostream & operator<<(std::ostream & os, plot_line_style p);

Description

Output description of plot_line_style. (mainly useful for diagnosis).

Example Usage: plot_line_style p; std::cout << p << std::endl; Outputs: point_line_style(RGB(0,0,0), blank, line, no bezier)
