Function operator<<

boost::svg::operator<<

Synopsis

// In header: <boost/svg_plot/detail/svg_elements.hpp>


std::ostream & operator<<(std::ostream & os, polygon_element & p);

Description

Output all poly_path_ points (May be useful for Boost.Test and diagnosis). ptr_vector<poly_path_point> poly_points; All the x, y coordinate pairs, Example: 

polygon_element p(1, 2, 3, 4, 5, 6);
std::cout << p << std::endl;

Outputs: (1, 2)(3, 4)(5, 6)
