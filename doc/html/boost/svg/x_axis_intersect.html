Type x_axis_intersect

boost::svg::x_axis_intersect — If and how the X axes intersects Y-axis.

Synopsis

// In header: <boost/svg_plot/detail/axis_plot_frame.hpp>



enum x_axis_intersect { bottom = = -1, x_intersects_y = = 0, top = = +1 };

Description

Note

x_axis_position_ and y_axis_position_ use x_axis_intersect & y_axis_intersect

bottom
X-axis free below bottom of end of Y-axis (case of all Y definitely < 0).
x_intersects_y
x_intersects_y when Y values include zero, so X intersects the Y-axis.
top
X-axis free above top of X-axis (case of all Y definitely > 0).
