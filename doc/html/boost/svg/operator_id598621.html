Function operator<<

boost::svg::operator<< — Output color to stream as RGB. See boost::svg::svg_color_constant.

Synopsis

// In header: <boost/svg_plot/svg_color.hpp>


std::ostream & operator<<(std::ostream &, const svg_color &);

Description

for example: "RGB(138, 43 , 226)" for blueviolet. This comment does not appear - for reasons entirely unclear.

Usage: svg_color my_color(127, 255, 212); cout << "my_color " << my_color << endl; Outputs: my_color RGB(127,255,212) std::cout << "magenta " << svg_color(magenta) << std::endl; but caution! 

std::cout << magenta << std::endl;

outputs int value 85 because magenta is an enum boost::svg::svg_color_constant !
