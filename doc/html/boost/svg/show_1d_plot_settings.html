Function show_1d_plot_settings

boost::svg::show_1d_plot_settings — Diagnostic display of all settings of a 1D plot. Outputs to a std::ostream a long list of about hundred of plot parameter settings, by default to std::cout. This list is invaluable if the plot does not look as expected.

Synopsis

// In header: <boost/svg_plot/show_1d_settings.hpp>


void show_1d_plot_settings(svg_1d_plot & plot, std::ostream & os = std::cout);

Description

Warning

This creates over 100 lines of output, so should be used sparingly!

Parameters:

os

std::ostream& for output, default std::cout.

plot

Name of svg_1d_plot.
