Function template isfinite

boost::math::isfinite — If a floating-point value is finite, return true.

Synopsis

// In header: <boost/svg_plot/detail/auto_axes.hpp>


template<typename FPT> bool isfinite(FPT t);

Description

Parameters:

t

floating-point value to test if finite.

Template Parameters:

FPT

floating-point type (float, double, long double, or user-defined).

Returns:

true if is finite, false if + or - infinite, or any NaN.
